# Black Falcons Management Principles Website

This is a static recreation of the Black Falcons Management Principles website.
It is built with HTML, CSS (Tailwind), and Vanilla JavaScript, requiring no build steps or external dependencies.

## Project Structure

- `index.html`: The main entry point.
- `css/`: Contains local font definitions.
- `js/`: Contains the logic (`main.js`) and libraries (`tailwindcss.js`, `lucide.js`).
- `assets/`: Contains static assets like fonts.

## How to Run

Simply open `index.html` in any modern web browser.
No server or installation is required.

## Features

- **Responsive Design**: Works on desktop and mobile.
- **Animations**: Scroll reveal, parallax, and typing effects.
- **Interactive Elements**: Mobile menu, tabs, copy link, download simulation.
- **Offline Capable**: All assets (fonts, scripts) are included locally.
